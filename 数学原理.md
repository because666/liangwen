## 📐 组合1：串行组合的数学原理

### 整体流程

```
输入数据 x
    ↓
【Mixup 数据增强】→ 混合样本 x̃
    ↓
【对抗训练】→ 生成对抗样本 x̃ₐdv
    ↓
【Focal Loss】→ 计算损失 L
    ↓
反向传播更新模型
```

---

## 1️⃣ Mixup 数据增强

### 1.1 基本公式

给定两个样本 和，混合生成新样本：

$$\tilde{x} = \lambda x_i + (1-\lambda) x_j$$

$$\tilde{y} = \lambda y_i + (1-\lambda) y_j$$

其中：
- $\lambda \sim \text{Beta}(\alpha, \alpha)$，从 Beta 分布采样
- $\alpha \in [0.1, 0.4]$ 通常取值

### 1.2 Beta 分布特性

Beta 分布的概率密度函数：

$$f(\lambda; \alpha, \beta) = \frac{\lambda^{\alpha-1}(1-\lambda)^{\beta-1}}{B(\alpha, \beta)}$$

当 $\alpha = \beta$ 时：
- $\alpha \to 0$：$\lambda$ 趋向于 0 或 1（几乎不混合）
- $\alpha = 1$：均匀分布
- $\alpha \to \infty$：$\lambda$ 趋向于 0.5（完全混合）

**对于 Mixup，通常取 $\alpha \in [0.1, 0.4]$**：
- $\alpha = 0.2$：$\lambda$ 大概率接近 0 或 1，轻微混合
- $\alpha = 0.4$：$\lambda$ 分布更均匀，中等混合

### 1.3 软标签处理

对于三分类问题，标签需要转换为 one-hot 编码：

$$y_i^{onehot} = \begin{cases} [1, 0, 0] & \text{if } y_i = 0 \\ [0, 1, 0] & \text{if } y_i = 1 \\ [0, 0, 1] & \text{if } y_i = 2 \end{cases}$$

混合后的软标签：

$$\tilde{y} = \lambda y_i^{onehot} + (1-\lambda) y_j^{onehot}$$

**例子**：
- $y_i = 0$（下跌），$y_j = 2$（上涨）
- $\lambda = 0.7$
- $\tilde{y} = 0.7 \times [1, 0, 0] + 0.3 \times [0, 0, 1] = [0.7, 0, 0.3]$

### 1.4 Mixup 的损失函数

使用 KL 散度或交叉熵：

$$L_{mixup} = -\sum_{c=0}^{2} \tilde{y}_c \log(p_c(\tilde{x}))$$

其中 $p_c(\tilde{x})$ 是模型对混合样本 $\tilde{x}$ 预测类别 $c$ 的概率。

## 2️⃣ 对抗训练

### 2.1 基本思想

在 Mixup 后的样本 $\tilde{x}$ 上添加对抗扰动 $\delta$：

$$\tilde{x}_{adv} = \tilde{x} + \delta$$

目标是找到最坏情况的扰动，然后让模型对这种扰动鲁棒：

$$\min_\theta \max_{\|\delta\| \leq \epsilon} L(f_\theta(\tilde{x} + \delta), \tilde{y})$$

### 2.2 FGSM（快速梯度符号法）

**单步攻击**，计算效率高：

$$\delta = \epsilon \cdot \text{sign}(\nabla_x L(f_\theta(\tilde{x}), \tilde{y}))$$

对抗样本：

$$\tilde{x}_{adv} = \tilde{x} + \epsilon \cdot \text{sign}(\nabla_x L)$$

**解释**：
- $\nabla_x L$：损失对输入的梯度，表示"哪个方向能让损失增加最快"
- $\text{sign}(\cdot)$：取符号，只保留方向
- $\epsilon$：扰动强度，控制步长

### 2.3 PGD（投影梯度下降）

**多步迭代攻击**，效果更好但计算成本高：

$$\tilde{x}_{adv}^{(0)} = \tilde{x} + \mathcal{U}(-\epsilon, \epsilon)$$

$$\tilde{x}_{adv}^{(t+1)} = \Pi_{\|\delta\| \leq \epsilon} \left( \tilde{x}_{adv}^{(t)} + \alpha \cdot \text{sign}(\nabla_x L(f_\theta(\tilde{x}_{adv}^{(t)}), \tilde{y})) \right)$$

其中：
- $\mathcal{U}(-\epsilon, \epsilon)$：均匀分布随机初始化
- $\Pi_{\|\delta\| \leq \epsilon}$：投影到 $\epsilon$-球内
- $\alpha$：每步步长
- $t = 1, 2, ..., T$：迭代次数

### 2.4 TRADES（权衡准确率和鲁棒性）

**分离自然损失和鲁棒损失**：

$$L_{TRADES} = L_{CE}(f_\theta(\tilde{x}), \tilde{y}) + \beta \cdot D_{KL}(f_\theta(\tilde{x}) \| f_\theta(\tilde{x}_{adv}))$$

其中：
- 第一项：自然样本的交叉熵损失（保持准确率）
- 第二项：自然样本和对抗样本预测的 KL 散度（提高鲁棒性）
- $\beta$：权衡参数

### 2.5 对抗训练的总损失

使用 FGSM 时：

$$L_{adv} = L(f_\theta(\tilde{x}), \tilde{y}) + L(f_\theta(\tilde{x}_{adv}), \tilde{y})$$

或使用 TRADES：

$$L_{adv} = L_{CE}(f_\theta(\tilde{x}), \tilde{y}) + \beta \cdot D_{KL}(f_\theta(\tilde{x}) \| f_\theta(\tilde{x}_{adv}))$$

## 3️⃣ Focal Loss

### 3.1 标准交叉熵

对于三分类问题，标准交叉熵：

$$L_{CE} = -\sum_{c=0}^{2} y_c \log(p_c)$$

其中 $y_c \in \{0, 1\}$ 是 one-hot 标签，$p_c$ 是预测概率。

### 3.2 Focal Loss 公式

$$L_{FL} = -\sum_{c=0}^{2} \alpha_c (1 - p_c)^\gamma y_c \log(p_c)$$

其中：
- $\alpha_c$：类别 $c$ 的权重（处理类别不平衡）
- $\gamma$：聚焦参数（降低易分类样本权重）
- $(1 - p_c)^\gamma$：调制因子

### 3.3 调制因子的作用

当样本容易分类时（$p_c \to 1$）：

$$(1 - p_c)^\gamma \to 0$$

当样本难分类时（$p_c \to 0$）：

$$(1 - p_c)^\gamma \to 1$$

**效果**：易分类样本的损失被抑制，难分类样本的损失被保留。

### 3.4 不同 $\gamma$ 的效果

| $\gamma$ | 易分类样本权重 ($p=0.9$) | 难分类样本权重 ($p=0.3$) | 比例 |
|---------|------------------------|------------------------|------|
| 0 | 1.00 | 1.00 | 1:1 |
| 1 | 0.10 | 0.70 | 1:7 |
| 2 | 0.01 | 0.49 | 1:49 |
| 5 | 0.00001 | 0.17 | 1:17000 |

### 3.5 类别权重 $\alpha_c$

根据类别频率设置：

$$\alpha_c = \frac{1}{N_c} / \sum_{c'} \frac{1}{N_{c'}}$$

其中 $N_c$ 是类别 $c$ 的样本数。

**对于我们的数据**：
- 下跌（$c=0$）：约 20%，$N_0 = 0.2N$
- 不变（$c=1$）：约 60%，$N_1 = 0.6N$
- 上涨（$c=2$）：约 20%，$N_2 = 0.2N$

计算：

$$\alpha_0 = \frac{1/0.2}{1/0.2 + 1/0.6 + 1/0.2} = \frac{5}{5 + 1.67 + 5} = 0.43$$

$$\alpha_1 = \frac{1/0.6}{11.67} = 0.14$$

$$\alpha_2 = \frac{1/0.2}{11.67} = 0.43$$

## 4️⃣ 组合后的总损失函数

### 4.1 完整公式

将三个技术组合：

$$L_{total} = L_{FL}(f_\theta(\tilde{x}), \tilde{y}) + L_{FL}(f_\theta(\tilde{x}_{adv}), \tilde{y})$$

展开：

$$L_{total} = -\sum_{c=0}^{2} \alpha_c (1 - p_c(\tilde{x}))^\gamma \tilde{y}_c \log(p_c(\tilde{x}))$$

$$- \sum_{c=0}^{2} \alpha_c (1 - p_c(\tilde{x}_{adv}))^\gamma \tilde{y}_c \log(p_c(\tilde{x}_{adv}))$$

### 4.2 训练流程伪代码

```python
for epoch in range(epochs):
    for batch_x, batch_y in dataloader:
        # Step 1: Mixup
        lam = np.random.beta(alpha, alpha)
        index = torch.randperm(batch_size)
        x_mix = lam * batch_x + (1-lam) * batch_x[index]
        y_mix = lam * batch_y + (1-lam) * batch_y[index]
        
        # Step 2: 对抗训练
        x_mix.requires_grad = True
        loss_nat = focal_loss(model(x_mix), y_mix)
        grad = torch.autograd.grad(loss_nat, x_mix)[0]
        x_adv = x_mix + epsilon * grad.sign()
        x_adv = torch.clamp(x_adv, x_mix - epsilon, x_mix + epsilon)
        
        # Step 3: 计算总损失
        loss_adv = focal_loss(model(x_adv), y_mix)
        loss_total = (loss_nat + loss_adv) / 2
        
        # 反向传播
        optimizer.zero_grad()
        loss_total.backward()
        optimizer.step()
```

## 5️⃣ 超参数选择

### 5.1 Mixup 参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| $\alpha$ | 0.2 ~ 0.4 | Beta 分布参数 |
| 混合概率 | 0.5 ~ 1.0 | 每个样本被混合的概率 |

### 5.2 对抗训练参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| $\epsilon$ | 0.001 ~ 0.01 | 扰动强度 |
| 攻击方式 | FGSM | 单步攻击，效率高 |
| 对抗权重 | 0.5 ~ 1.0 | 对抗损失的权重 |

### 5.3 Focal Loss 参数

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| $\gamma$ | 2.0 | 聚焦参数 |
| $\alpha_0$ | 0.43 | 下跌权重 |
| $\alpha_1$ | 0.14 | 不变权重 |
| $\alpha_2$ | 0.43 | 上涨权重 |

## 6️⃣ 数学直觉总结

### Mixup 的作用

$$\text{原始数据} \xrightarrow{\text{Mixup}} \text{连续标签空间}$$

- 让模型学习"中间状态"
- 决策边界更平滑
- 防止过拟合

### 对抗训练的作用

$$\text{平滑边界} \xrightarrow{\text{对抗训练}} \text{更宽的边界}$$

- 让模型对扰动鲁棒
- 防止模型记住噪声
- 提高泛化能力

### Focal Loss 的作用

$$\text{所有样本} \xrightarrow{\text{Focal Loss}} \text{关注难样本}$$

- 降低易分类样本权重
- 提高难分类样本权重
- 处理类别不平衡

### 组合效果

$$\text{过拟合模型} \xrightarrow{\text{三重正则化}} \text{鲁棒模型}$$

- 验证集准确率可能下降
- 评测集准确率期望提升
- 泛化能力增强
