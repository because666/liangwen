# 架构V3 - 深度收益导向分层集成网络

## 版本信息
- 创建日期: 2025-04-08
- 状态: 已归档（训练不稳定，效果不佳）

## 架构概述

### 核心设计
- **4个异构专家**: T-KAN、Level Attention、Patch Transformer、Cross-Asset OFI
- **动态门控融合**: 根据市场状态动态分配专家权重
- **收益导向损失**: 收益加权 CE + Sharpe Ratio + 稀疏正则
- **直接三分类**: 不使用动作门，直接预测 0/1/2

### 参数量
- 总参数: ~50 万

## 遇到的问题

### 1. 训练不稳定
- Loss 经常出现 NaN
- 梯度爆炸
- 收益率极端值

### 2. 效果不佳
- 提交评测得分差
- 可能原因：模型过于复杂，数据量不足

## 经验教训

1. **模型复杂度 vs 数据量**: 数据量不足以支撑复杂模型
2. **多专家融合**: 梯度冲突，训练困难
3. **复合损失**: 优化目标冲突
4. **数值稳定性**: B-spline 对数值敏感

## 后续方向

转向更简单的架构：
- T-KAN 单一模型
- 简化损失函数
- 减少参数量
